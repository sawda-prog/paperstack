# PaperStack

Free past papers downloader. No API key needed, no costs, just deploy and share the URL.

## Deploy on Render (free, 5 minutes)

1. Push this folder to a GitHub repo
2. Go to render.com → sign in with GitHub
3. Click New → Web Service → select your repo
4. Render auto-detects Python — just click Deploy
5. Share the URL with your team

That's it. Free forever.

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open http://localhost:5000
